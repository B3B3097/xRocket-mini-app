# 🚀 XRocket AI Trader - Публичный сайт (Mini App)

Этот репозиторий содержит фронтенд для Telegram Mini App, который подключается к бэкенду на Replit.

## 📁 Структура

- `xrocket_public_site/index.html` - Основной файл мини-приложения
- `.github/workflows/deploy-public-site.yml` - Автоматический деплой на GitHub Pages

## ⚙️ Настройка

### 1. Обновление URL бэкенда

Откройте файл `xrocket_public_site/index.html` и найдите строку:

```javascript
const BACKEND_URL = "https://45bf33ee-f2b3-4074-9c7d-74c132598100-00-38bwv545h4jni.spock.replit.dev:5000";
```

Замените URL на актуальный адрес вашего приложения на Replit.

### 2. Включение GitHub Pages

1. Перейдите в **Settings** → **Pages**
2. В разделе **Build and deployment**:
   - **Source**: Выберите `GitHub Actions`
3. Сохраните изменения

### 3. Автоматический деплой

После каждого пуша в ветку `main`, сайт будет автоматически обновляться через GitHub Actions.

## 🔗 Подключение к Telegram

1. Запустите бота через @BotFather
2. Используйте команду `/newapp`
3. Укажите ссылку на ваш GitHub Pages сайт (будет видна после деплоя)
4. Готово!

## 🔐 Безопасность

- API ключи НЕ хранятся в этом репозитории
- Ключи вводятся пользователем в мини-аппе и отправляются напрямую на бэкенд (Replit)
- Ключи удаляются из памяти сервера через 4 часа

## 📱 Функции Mini App

- 📊 График цены в реальном времени
- 🤖 AI сигналы (BUY/SELL/WAIT)
- 💬 Чат с AI помощником
- 🔄 Переключение Демо/Реальный режим
- 🔑 Безопасный ввод API ключей
- 📈 Статистика торговли

**Фронтенд**: Хостится на GitHub Pages (HTML + JS)
